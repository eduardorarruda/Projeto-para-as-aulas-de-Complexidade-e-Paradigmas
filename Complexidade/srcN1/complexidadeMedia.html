Complexidade de Algoritmo

Complexidade Média

A complexidade média de um algoritmo é uma média ponderada do desempenho de cada entrada pela probabilidade de ela ocorrer. Dado um algoritmo a, considere: O conjunto Dn = { d ∈ D / tam( d ) = n } de suas entradas com tamanho n; a probabilidade prob( d ) de ocorrer a entrada d de Dn, o desempenho desemp[ a ](d) = custo( exec[ a ]( d ) ) de a com entrada d.

cM[a](n) := ∑ [prob(d) · desemp[a](d)]

Onde Dn = {d ϵ D / tam (d) = n} Prob (d) é a probabilidade de ocorrer a entrada d de Dn Desem P[a] (d) = custo (exec[a](d)) do algoritmo a com a entrada d.

QUICK SORT

Função Quicksort (A : V, m, p : Nat) -> A : V {ordenação concatenação}

{Entrada : Vetor A [m . . . p] (a ordenar)

SAÍDA: Vetor A:V (o vetor de entrada em ordem crescente)}

    Sorte V:= Vetor [m . . . p]
    1. k ← ind( pivô, A , m, p );
    2. se k = 0 3. então retorne-saída( A )
    4. senão 5. Troca( A[ m ], A[ k ] );
    6. Particiona( A, m, p, j );
    7. L’ ← Quicksort( A, m, j – 1 );
    8. L” ← Quicksort( A, j, p );
    9. L ← Concat( L’, L” );
    10. retorne-saída( A )
    11. fim-senão
    12. fim-se;
    13. fim-Função
